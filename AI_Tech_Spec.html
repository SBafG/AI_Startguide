Teknisk spec

Vad du behöver för att bygga en AI-portal

Komponenter, verktyg och söktermer — för en analytiker som vill komma igång

Det här är en praktisk komponentlista för den som vill bygga ett AI-drivet analysstöd — typ en intern portal där data hämtas, AI bearbetar den och resultatet visas. Varje komponent har en sökterm du kan googla på för att lära dig mer.

Du behöver inte allt på en gång. Börja med de tre som är märkta NÖDVÄNDIG — resten lägger du till när du ser behovet.

🖥️ Utvecklingsmiljö

VS Code
GratisNödvändig
Din arbetsyta. Textredigerare från Microsoft där du skriver och kör kod. Kan byggas ut med tillägg (extensions). Fungerar för HTML, JavaScript, Python och mer.
Visual Studio Code download
Claude Code
BetaldNödvändig
AI-assistent som sitter direkt i VS Code. Du beskriver vad du vill bygga — Claude skriver koden, förklarar den och felsöker. Det är det här verktyget som gör att du kan bygga utan att kunna programmera.
Claude Code VS Code extension Anthropic
Node.js
GratisNödvändig
Kör JavaScript-kod på din dator (utanför webbläsaren). Behövs för att läsa filer, anropa API:er, schemalägga jobb och bygga backend-scripts. Installeras en gång, används hela tiden.
Node.js download LTS
Git + GitHub
GratisRekommenderas
Versionshantering — sparar historik på alla ändringar och låter dig återställa om något går fel. GitHub är molntjänsten där koden lagras. Används också för att publicera webbsidor via GitHub Pages.
Git for Windows download

🤖 AI-modeller & API

Anthropic API (Claude)
Betald per användningNödvändig
Det är via detta API ditt script pratar med Claude. Du betalar per token (ungefär per ord). En månadsrapport kostar under 1 kr. Välj modell efter behov: Haiku = snabb & billig, Sonnet = balanserad, Opus = mest kraftfull.
Anthropic API console Claude models pricing
Microsoft Copilot
Microsoft 365Tillval
AI inbyggd i Word, Excel, Teams och Outlook. Bra för enstaka dokumentuppgifter direkt i Office — men ger dig inte samma kontroll som ett eget script mot Claude API. Kräver Copilot-licens (~4 400 kr/år/användare).
Microsoft 365 Copilot licens pris

🗄️ Data & lagring

SharePoint / SharePoint Lists
Microsoft 365Rekommenderas
Microsofts dokumentlagring och listtjänst. Fungerar utmärkt som databas för portalen — t.ex. för att lagra ärenden, kunskapsartiklar eller rapporter. Kan läsas och skrivas via Graph API. Ingår i de flesta M365-licenser.
SharePoint Lists REST API Microsoft Graph
CSV / Excel-export
Gratis
Den enklaste datakällan — en fil som ditt portföljsystem eller annat internsystem exporterar. Scriptet läser filen direkt. Ingen databas, ingen integration behövs för att komma igång.
Node.js read CSV file parse

⚙️ Automatisering & triggers

Power Automate
Microsoft 365Rekommenderas
Microsofts automatiseringsverktyg — bygger flöden utan kod. Kan triggas av en ny fil i SharePoint, ett mail, ett schema eller en HTTP-förfrågan. Bra för att koppla ihop Office-tjänster med ditt script.
Power Automate HTTP trigger SharePoint flow
PM2
GratisRekommenderas
Processhanterare för Node.js. Låter dina scripts köra som bakgrundstjänster — startar om automatiskt vid fel och efter omstart av datorn. Enklaste sättet att hålla en filbevakning igång.
PM2 process manager Node.js background service
Windows Task Scheduler
Gratis · Inbyggd i Windows
Schemalägg ett script att köras vid en viss tid — t.ex. varje måndag 08:00. Inget att installera. Enklaste sättet att automatisera utan tredjepartsverktyg.
Windows Task Scheduler run script scheduled

🌐 Publicering & portal

Azure Static Web Apps
Gratis nivå finnsRekommenderas
Microsofts hosting för statiska webbsidor. Kopplas direkt till GitHub — varje push till main-branchen deployas automatiskt på ~2 minuter. Stödjer Microsoft-inloggning (Entra ID) så att portalen skyddas bakom era M365-konton.
Azure Static Web Apps GitHub deploy Entra ID authentication
GitHub Pages
Gratis
Enklaste sättet att publicera HTML-sidor offentligt — precis som den här guiden. Inga inloggningsskydd, men perfekt för demos och guider utan känslig data. Driftsätts automatiskt från GitHub-repot.
GitHub Pages enable publish HTML

🔧 Avancerat — när du är redo

MCP — Model Context Protocol
Öppen standardAvancerat
Ett protokoll från Anthropic som låter Claude prata direkt med externa system — t.ex. din databas, ditt portföljsystem eller ett internt API. Gör att AI kan hämta och skriva data i realtid utan att du manuellt exporterar filer.
Model Context Protocol MCP Anthropic Claude tools
Microsoft Graph API
Microsoft 365Avancerat
API som ger ditt script tillgång till hela M365 — läsa SharePoint-listor, hämta användarinfo från Entra ID, skicka mail via Outlook, läsa Teams-meddelanden. Standardsättet att integrera med Microsofts ekosystem.
Microsoft Graph API JavaScript SDK getting started
